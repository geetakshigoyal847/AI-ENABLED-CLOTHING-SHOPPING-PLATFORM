StyleNest – AI-Powered Fashion E-Commerce Website

StyleNest is a full-stack online clothing shopping platform developed using PHP and MySQL. The website provides a complete e-commerce experience with product browsing, user authentication, shopping cart, checkout, and order management features.

Features

• User registration and login system
• Admin and customer role-based access
• Product catalogue with categories
• Search and filtering functionality
• Shopping cart and checkout system
• Order placement and customer order history
• Admin dashboard management
• MySQL database integration
• Responsive fashion-focused user interface

Technologies Used

PHP
MySQL
HTML5
CSS3
JavaScript

Project Overview

StyleNest demonstrates the development of a business information system by combining web development, database management, and e-commerce functionality to create a live online fashion shopping platform.
